Skills & Experências

Design de Interação

Um Design de qualidade em grande parte influência a tomada de decisão de um usuário. E uma ótima UX tem tudo haver com contar histórias, quanto melhor a história é contada, melhor a conversão de um website/app.

  • Wireframes
  • Prototipação: Adobe XD, Balsamiq Mockups
  • Design Responsivo
  • Planejamento de Conteúdo
  • Princípios de Usabilidade e Acessibilidade

Desenvolvimento Web

Escrever código que esteja pronto para ir para a produção e que funcione em diversos dispositivos se tornou uma padrão para alcançar regras de negócio e necessidades do usuário, de forma que também sigam as melhores práticas de desenvolvimento web.

  • HTML 5
  • CSS 3 Grid and Flexbox, BEM, pré-processadores (Sass) e Bootstrap
  • Javascript moderno: ES6+ Arrow Functions, Módulos, Closures and Promises
  • Cliente: React, MeteorJS e AngularJS
  • Servidor: Node.js GraphQL, e PHP (CakePHP)
  • Banco de dados: GraphQL, MySQL e MongoDB
  • Integração de APIs

Desempenho & Métricas

Tempo de carregamento nunca foi mais importante. Após 3 segundos de espera, visitantes já estão deixando um website. Portanto, uma abordagem que é focada no desempenho na hora de desenvolver é essencial na hora de otimizar o engajamento de usuários.

  • Google Analytics
  • Optmização dos mecanismos de busca (SEO)
  • Aplicações Web Progressivas (PWA)
  • Experiências offline (Service Workers)

Testes & Ferramentas

Desenvolvimento Orientado a Testes (TDD) é uma prática de desenvolvimento que reduz o número de bugs que vazam durante a fase de deploy, fazendo com que seja crítico para entregar aplicações efetivas para o Mercado.

  • frameworks de Teste: Jest, Mocha
  • npm, Webpack and Gulp
  • VCS: Git, GitHub, GitLab
  • IDE: Visual Studio Code, Atom
  • Slack, Skype, Trello
